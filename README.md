## Running Locally

```bash
git clone https://github.com/adityabansal-tech/Portfolio.git
cd Portfolio
npm install
npm start
```
Opens at `http://localhost:3000`.

## Deployment

```bash
npm run deploy
```
Builds and publishes to GitHub Pages via the `master` branch.
